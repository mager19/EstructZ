  • HTML Estructura

  • Estructura

    La estructura esta basada en la grid de bootstrap framework usado para el diseño del sitio, que consta de 12 columnas mediante las cuales se va desarrollando de manera fluida y vertical. Se ha etiquetado mediante clases, dentro de un contenedor principal, luego de este una clase row que alberga el contenido, y dentro de ello la clase normal de cada elemento.



  • Css

  • Estructura

    Se usa un archivo style.css central, donde se manejan las clases nombradas en los archivos html. Tiene comentarios de inicio de cada clase que hará fácil encontrar las clases necesarias.
